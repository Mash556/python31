что такое протокол ?
протокол это набор правил о котором устройства и программы обмениваются данными

HTTP - используется для передачи произвольных данных (HyperText Transfer Protocol)



методы HTTP:
GET - получение ресурса
POST -  создание ресурса (отправка данных)
PUT - полная замена ресурса
PUTCH - частичная замена ресурса
DELETE - удаляет ресурса
HEAD - запрашивает загаловки
OPTIONS - получение списка допустимых методов



url -  UNIFORM RESOURSE LOCATOR (унифицированный определитель местонахождение ресурса)
uri - Uniform Resourse Identifier
urn - Uniform Resourse Name




cостояние HTTP (5 классов 1 - 5)
1хх - информационные
2хх - успешный запрос
200 - ok
201 - Created
204 - No kontent

3хх - перенаправление
4хх - ошибки клиента
404 - Not Found
401 - Unauthoried
400 - Bad Bequests


5хх - ошибка сервера
500 - Internal Server Error
